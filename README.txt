项目:云笔记(cloud_note)
软件环境:myeclipse10.6
系统环境:win10
数据库:mysql

功能:
具有登录注册功能,新建笔记本以及新建笔记功能,
分享笔记管理,收藏笔记管理,回收站管理,活动管理.

项目心得:
主要用SSM框架来搭建云笔记,在编写代码过程中,总体上要知道大概的过程,
具体细节只有在编写调试过程中调整,客户端主要用jquery,用ajax发出请求,
服务端用Tomcat容器,用json格式来通过Controller来响应客户端.数据库主要
用dao来进行增删改查.

责任描述:
负责实现登录注册功能以及其他功能,让客户端和服务端进行互动,
调用数据库.

目前功能还在完善中,有些功能存在bug,本人的代码写作能力
见拙了,还望各位多多包涵,欢迎各位指出代码不足,诚心请教.
